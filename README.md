# منصة إدارة المشاريع مع إدارة الفريق والمهام والوقت
==============================================

## نظرة عامة وغرض المشروع
------------------------

منصة إدارة المشاريع هي نظام إدارة شاملة يهدف إلى تسهيل إدارة المشاريع من البداية إلى النهاية. يتيح للمستخدمين إنشاء وتنظيم وتحديد الأولويات للمهام، وتعيين المهام إلى أعضاء الفريق، وتتبع التقدم، وإدارة الوقت. يهدف المشروع إلى توفير أداة قوية ومرنة للمشرفين على المشاريع لزيادة الإنتاجية وتحسين التعاون بين أعضاء الفريق.

## هيكل المشروع
----------------

*   `/app`: يحتوي على كود التطبيق الرئيسي
*   `/config`: يحتوي على إعدادات التطبيق
*   `/database`: يحتوي على قاعدة البيانات
*   `/docker`: يحتوي على ملفات Docker
*   `/requirements`: يحتوي على متطلبات التطبيق

## تشغيل البيئة باستخدام docker-compose
--------------------------------------

1.  قم بتثبيت Docker وdocker-compose على جهازك.
2.  افتح Terminal وانتقل إلى دليل المشروع.
3.  قم بتشغيل الأمر `docker-compose up -d` لتشغيل الحاويات في الخلفية.
4.  قم بتشغيل الأمر `docker-compose exec web python manage.py migrate` لتنفيذ عمليات الهجرة.
5.  قم بتشغيل الأمر `docker-compose exec web python manage.py runserver` لتشغيل خادم التطوير.
6.  افتح متصفحك وانتقل إلى `http://localhost:8000` للوصول إلى التطبيق.

## قائمة الوحدات والجداول والأدوار
---------------------------------

### الوحدات

*   `projects`: وحدة إدارة المشاريع
*   `teams`: وحدة إدارة الفريق
*   `tasks`: وحدة إدارة المهام
*   `time`: وحدة إدارة الوقت

### الجداول

*   `projects_project`: جدول المشاريع
*   `teams_team`: جدول الفريق
*   `tasks_task`: جدول المهام
*   `time_time`: جدول الوقت

### الأدوار

*   `admin`: دور المشرف
*   `manager`: دور المدير
*   `team_member`: دور عضو الفريق
